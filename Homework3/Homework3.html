PHP та JavaScript

Мови програмування PHP PHP і JavaScript дозволяють зробити сайт динамічним, тобто таким, що реагує на дії користувача. Наприклад можна зробити гарне меню, що випадає, або слайдер.

Види скриптів

Для цього пишуться скрипти (англ. script -"сценарій") програми, що дозваляютьреагувати на дії користувача. Скрипти бувають двох видів.

  • ті, які виконуються на сервері, а результат їхнього виконання приходить у браузер до користувача вже в готовому вигляді. це скрипти, написані мовою PHP. На ньому пишуться CMS-ки- системи керування контентом.
  • ті які виконуються прямо у браузері користувача. Це скрпти ,написані мовою JavaScript. Вони найчастіше використовуються для того, щоб зробити сторінку більш зручною та красивою.
